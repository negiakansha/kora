# Inputs

1. Cameras
    - Live Feed Object Recognition
    - Facial Recognition
2. Mics
    - Wake Word: "Kora"
    - STT API
3. Inertial Mass Unit (IMU)
    - Gyro Sensors
    - Accelerometers
    - Magnetometers
4. Touch Sensor
    - Touch Sensor on top of head for head pats

> All Data is used to produce commands specified in `Commands.md`
